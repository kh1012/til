# interface

- type alias와 조금 차이난다.
- 개발에서는 일관성이 있어야 한다.
- 코드를 읽는다의 관점에서 사용한다.
- 상속관계를 만들때에는 표현한다.

```ts
interface News {
  ...
}

interface NewsFeed extends News {
  readonly id: number;
  ...
}
```
