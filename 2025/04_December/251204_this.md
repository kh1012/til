# javascript의 this

this 자체는 호출되는 위치에서 실행 컨텍스트의 범위와 렉시컬 환경에 따라 의미 단위가 달라질 수 있다.  
기본적으로 브라우저에서는 window, node에서는 global을 일 컫는다.  
class에 정의된 메서드라면, 자기 자신의 인스턴스를 가리킨다.  
callback 역시, window / global을 가리킨다. 단, 인자로 넘기면 인스턴스를 지칭할수도 있게된다.  
머리에 모든 경우가 담기진 않았지만, 기본은 window이고 인스턴스일 경우 자기 자신을 가리킨다는 점을 인지하는걸로!

| 위치           | 의미 단위               |
|--------------|---------------------|
| 전역공간에서       | window / global     |
| 함수 호출시       | window / global     |
| 메서드 호출시      | 메서드 호출 주체 (메서드 명 앞) |
| callback 호출시 | 기본적으로는 함수내부에서와 동일   |
| 생성자 함수 호출시   | 인스턴스                |
